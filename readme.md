JavaScript Calculator

A simple and functional calculator built using HTML, CSS, and JavaScript.  
It keyboard Enter key for calculation.


Features

1. Basic arithmetic operations: (+ , -,   * , /)
2. Percentage (%) support
3. Clear all (Ac) and delete one character (del)
4. Keyboard Enter key support
5. Prevents incorrect results after pressing Enter
6. Handles division by zero and invalid expressions
